Internal implementation types that power the Corvus.Text.Json runtime. These include the document storage layer (`MetadataDb`, `ComplexValueBuilder`, `IJsonDocument`), element interfaces (`IJsonElement`, `IMutableJsonDocument`), enumerators, JSON Schema evaluation machinery, and code-generation support types.

These types are public for use by generated code but are not intended to be used directly by application developers. Breaking changes may occur between minor versions.
