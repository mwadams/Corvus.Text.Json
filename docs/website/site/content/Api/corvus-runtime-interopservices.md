Low-level marshalling utilities for direct access to the internal memory layout of JSON documents. [`JsonMarshal`](/api/corvus-runtime-interopservices-jsonmarshal.html) provides methods for extracting raw UTF-8 property names and values. These are advanced APIs intended for performance-critical interop scenarios.

| Type | Kind | Description |
|------|------|-------------|
| [JsonMarshal](/api/corvus-runtime-interopservices-jsonmarshal.html) | class | An unsafe class that provides a set of methods to access the underlying data representations of JSON types. |
